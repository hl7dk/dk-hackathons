# DK Hackathons 1.0.0

This is the first release of the Danish Hackathon Implementation Guide, published by HL7 Denmark for the Danish Hackathon 2026.

The IG supports the health data hackathon scheduled for October 6, 2026 at Forskerparken 10, Odense, Denmark, the day before eSundhedsobservatoriet. It is part of the Nordic Health Data Hackathons series and provides participant guidance, track material, and FHIR R4 artifacts for hands-on work with Danish health data interoperability.

## Hackathon Content

This release includes:

- Home, preparation, schedule, and results pages for the Danish Hackathon 2026.
- Preparatory webinar material for the September 22 and September 29, 2026 sessions.
- Track 1: Agentic Patient Access.
- Track 2: Telemedicine.
- A placeholder page for Track 3, which is still to be announced.
